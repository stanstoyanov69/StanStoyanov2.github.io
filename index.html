Analyzing Unicorn Companies

As part of this project, I analyzed trends in high-growth companies for an investment firm. The goal was to identify industries producing the highest valuations and understand the emergence of new high-value companies. This analysis provides valuable insights into industry trends and informs the firm's future portfolio strategy.

The analysis was performed using data from the unicorns database, which includes information from various tables such as "dates," "funding," "industries," and "companies." The project involved the following key tasks:

  • Identified the top three performing industries.
  • Calculated the number of unicorn companies and the average valuation for each industry by year (2019, 2020, 2021).
  • Generated a report summarizing the findings in a structured table format.
